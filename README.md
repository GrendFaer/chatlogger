# ChatLogger

Мод для **Minecraft Fabric 1.21.4** для сбора информации об игроках из чата с группировкой по кланам.

## 📋 Возможности

### Основные функции
- **Автоматический сбор данных** — парсинг сообщений чата для извлечения информации об игроках (ник, клан, уровень)
- **Группировка по кланам** — организация игроков по кланам с возможностью экспорта
- **Подсветка игроков** — визуальная подсветка игроков из кланов с закреплённым цветом
- **Проверка онлайна** — проверка статуса игроков через личные сообщения
- **Отправка в клан-чат** — быстрая отправка состава клана в клановый чат
- **Фильтрация ников** — автоматическое отклонение некорректных ников (числовые, слишком длинные)

### ✨ Новые функции (v1.0)
- **Авто-сообщения** — автоматическая отправка сообщений по таймеру или горячей клавише (`M`)
- **Руны** — быстрый выбор и применение рун через GUI (`R`)
- **Переключатель мечей** — мгновенное переключение между мечами (`V`)
- **Быстрые команды для стаффа** — шаблоны действий для модерации (`B`)
- **GUI экраны** — удобное управление через графический интерфейс
- **Локализация** — полная поддержка русского и английского языков

## 📥 Установка

### Быстрая установка (скачать готовый мод)

1. **Скачайте JAR файл мода:**
   - Перейдите на страницу [Releases](https://github.com/GrendFaer/chatlogger/releases)
   - Скачайте последний файл `chatlogger-*.jar`
   - Или напрямую: [Скачать JAR](https://github.com/GrendFaer/chatlogger/releases/latest)

2. **Установите Fabric Loader и Fabric API** (см. ниже)

3. **Поместите JAR файл в папку mods:**
   - `%appdata%/.minecraft/mods/` (Windows)
   - `~/Library/Application Support/minecraft/mods/` (Mac)
   - `~/.minecraft/mods/` (Linux)

### Требования
- Minecraft 1.21.4
- [Fabric Loader](https://fabricmc.net/use/installer/) 0.17.3 или выше
- [Fabric API](https://modrinth.com/mod/fabric-api) 0.119.4+1.21.4 или выше

### Шаги установки

1. **Установите Fabric Loader**
   - Скачайте установщик с https://fabricmc.net/use/installer/
   - Запустите и выберите версию Minecraft **1.21.4**
   - Нажмите "Install"

2. **Скачайте мод**
   - Перейдите на страницу [Releases](https://github.com/GrendFaer/chatlogger/releases)
   - Скачайте последний `.jar` файл
   - Или соберите мод самостоятельно (см. раздел "Сборка")

3. **Установите Fabric API**
   - Скачайте с https://modrinth.com/mod/fabric-api/versions?g=1.21.4
   - Поместите файл в папку `mods`

4. **Поместите мод в папку mods**
   - Скопируйте `chatlogger-*.jar` в `%appdata%/.minecraft/mods/`
   - Или в `run/.minecraft/mods/` для среды разработки

5. **Запустите Minecraft**
   - Выберите профиль Fabric 1.21.4
   - Запустите игру
   - При загрузке должно появиться сообщение: `[ChatLogger] Мод активен!`

## 🔧 Сборка из исходников

### Требования
- Java 21 или выше
- Git

### Шаги сборки

```bash
# Клонируйте репозиторий
git clone https://github.com/GrendFaer/chatlogger.git
cd chatlogger

# Соберите мод
gradlew.bat build          # Windows
./gradlew build            # Linux/Mac

# Готовый JAR файл будет в build/libs/
```

## ⌨️ Команды

### Основные команды

| Команда | Описание |
|---------|----------|
| `/chatlogger export [клан]` | Экспортировать данные в файл |
| `/chatlogger list [клан]` | Показать список игроков |
| `/chatlogger clans` | Показать список всех кланов |
| `/chatlogger count` | Показать количество игроков |
| `/chatlogger clear` | Очистить все данные |
| `/chatlogger clean` | Удалить неверные ники (числовые) |
| `/chatlogger check [клан]` | Проверить онлайн игроков |
| `/chatlogger stop` | Остановить проверку онлайна |

### Управление кланами

| Команда | Описание |
|---------|----------|
| `/chatlogger add <игрок> <клан> [уровень]` | Добавить игрока в клан |
| `/chatlogger remove <игрок>` | Удалить игрока из клана |

### Настройки

| Команда | Описание |
|---------|----------|
| `/chatlogger config` | Показать настройки |
| `/chatlogger config message <текст>` | Изменить сообщение для проверки |
| `/chatlogger config command <команда>` | Изменить команду для ЛС |
| `/chatlogger config reset` | Сбросить настройки |

### Подсветка кланов

| Команда | Описание |
|---------|----------|
| `/chatlogger highlight` | Показать статус подсветки |
| `/chatlogger highlight enable` | Включить подсветку |
| `/chatlogger highlight disable` | Выключить подсветку |
| `/chatlogger highlight color <клан> <цвет>` | Закрепить цвет за кланом |
| `/chatlogger highlight remove <клан>` | Убрать цвет клана |
| `/chatlogger highlight list` | Показать все цвета кланов |
| `/chatlogger highlight debug <игрок>` | Отладка подсветки |

### Клан-чат

| Команда | Описание |
|---------|----------|
| `/chatlogger sendclan <клан>` | Отправить состав клана в клан-чат |

### Новые функции (v1.0)

| Команда | Описание |
|---------|----------|
| `/chatlogger automessage` | Управление авто-сообщениями |
| `/chatlogger runes` | Управление рунами |
| `/chatlogger swordswitcher` | Управление переключателем мечей |
| `/chatlogger config` | Открыть главное меню настроек |

## 🎨 Цвета для подсветки

Поддерживаемые названия цветов:
- `white`, `red`, `green`, `blue`, `yellow`, `cyan`, `magenta`, `black`, `gray`
- `orange`, `pink`, `purple`, `lime`, `teal`, `navy`, `brown`, `gold`, `silver`
- Русские названия: `белый`, `красный`, `зелёный`, `синий`, и т.д.
- HEX формат: `#FF0000`, `#00FF00`, и т.п.

**Пример:**
```
/chatlogger highlight color WHITE red
/chatlogger highlight color CLOWNMEW #FF5500
/chatlogger highlight color KABЫ зелёный
```

## ⚙️ Настройка

### Изменение команды для ЛС
По умолчанию используется команда `/m`. Для изменения:
```
/chatlogger config command tell
```

### Изменение сообщения для проверки
По умолчанию сообщение `1`. Для изменения:
```
/chatlogger config message check
```

### Символ для клан-чата
По умолчанию используется `@`. Для изменения отредактируйте `chatlogger_config.properties`:
```
clanChatSymbol=@
```

## 🐛 Решение проблем

### Мод не загружается
1. Убедитесь, что установлен Fabric Loader 1.21.4
2. Убедитесь, что установлен Fabric API
3. Проверьте логи в `.minecraft/logs/latest.log`

### Игроки не подсвечиваются
1. Проверьте, что подсветка включена: `/chatlogger highlight list`
2. Убедитесь, что цвет установлен: `/chatlogger highlight color <клан> <цвет>`
3. Проверьте игрока: `/chatlogger highlight debug <ник>`
4. Убедитесь, что игрок есть в списке: `/chatlogger list <клан>`

### Ошибка "String too big" при отправке в клан-чат
Сообщение обрезается автоматически до 256 символов (лимит Minecraft). Используйте более короткие ники или отправляйте по частям.
